Aplicativo híbrido de gestão financeira pessoal baseado na metodologia de envelopes (orçamento base zero). Desenvolvido com React Native (Expo) como trabalho de avaliação para a disciplina de Desenvolvimento de Aplicativos Híbridos.

**Principais Recursos:**

- **Autenticação:** Acesso seguro aos dados financeiros via Biometria (```expo-local-authentication```).

- **Gestão Offline:** Armazenamento do saldo das categorias e histórico utilizando ```AsyncStorage```.

- **Captura de Comprovantes:** Integração com a Câmera (```expo-camera```) para anexar fotos de recibos às transações.

- **Mapeamento de Gastos:** Uso de GPS e Mapas (```expo-location``` e ```react-native-maps```) para registrar o local das compras.

- **Sincronização Nuvem:** Integração com API REST via ```Axios```.
